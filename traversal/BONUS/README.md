## Contents

### TSP (Traveling Salesperson Problem)

A classic NP-hard optimization problem. Implement a brute-force DFS solution to find the shortest route visiting all cities exactly once.

**Status:** Incomplete - has TODOs for DFS implementation  
**What's provided:** City data structure, distance calculations, JSON city loader  
**Your task:** Complete the `solveTSPBruteforce()` function using DFS

**To work on it:**
```bash
cd TSP
make
./tsp_solver  # Once implemented
```
